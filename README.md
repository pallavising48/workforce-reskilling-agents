# workforce-reskilling-agents
Multi-agent workforce retraining system using Microsoft Foundry.
Skill portability + capacity-aware planning. Example: Support reps spend 6 hours a day understanding why customers are frustrated. AI ops engineers spend 6 hours understanding why AI agents are behaving unexpectedly. Same debugging pattern, different domain.
For this submission focus: Support reps → AI operations. 

Data sources:
Hours available: Derived from Work IQ analysis of calendar (focus time, meeting load)
Learning style: Inferred from historical L&D engagement patterns
Confidence level: Based on past skill assessments and product mastery scores
[This synthetic data simulates what those systems would return in production.]
